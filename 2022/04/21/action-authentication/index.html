摘要

​ 在大数据时代飞速发展的背景下，身份认证已经渗透到生活中的各个领域，在电子医疗、金融业务和社交网络等行业中发挥着无比重要的作用。但是随着不法分子攻击手段的不断进步，传统的认证方式面临着愈发严峻的形势和挑战，对用户的隐私信息安全和财产安全造成了严重威胁。为了防御攻击，传统认证中会加入更加繁琐的内容，使用户在认证过程中花费较多的时间和精力，因此当前迫切需要一种安全、便捷的身份认证方式。

​ 本文介绍了一种基于音乐敲击节奏的身份认证方式，在原有的认证方式上大胆创新，将基于某段音乐的敲击节奏作为用户认证“密码”。与传统认证方式相比，基于音乐敲击节奏的身份认证方式使用经济实惠的设备进行数据收集，并且该设备可独立于其他设备，是实现跨设备进行身份认证的硬件基础。除此之外，使用相同音乐的敲击节奏，不同用户有自己独特的感受和理解，在抵御常见的暴力攻击和不法分子的模仿时表现出令人满意的可用性。

​ 这篇文章主要讲述了基于音乐敲击节奏的身份认证方式的数据收集、数据分析、特征提取和认证匹配等部分，经过性能分析和安全测试表明：该方式简单便捷，安全性较高，可抵御各种类型的攻击，具有极高的经济实用价值和开发推广潜力。

​ 关键词：身份认证 节奏敲击

第一章 作品概述

1 背景

1.1 背景分析

​ 随着科技的进步，VR虚拟现实技术得到了快速发展，并且不断受到欢迎。目前，VR技术正在迅速渗透到各种关键任务应用程序中，包括电子商务、医疗保健、社交聊天、军事训练和线上教育等。在这些应用程序中，VR设备中存储着用户的个人信息，如电子邮件、照片、视频以及登录密码等，如果不法分子窃取到这些隐私信息可能会利用其做违法犯罪行为，造成各方无法挽回的损失。因此用户认证机制的使用对于保障VR设备的使用安全至关重要，我们必须保证只有所有者及授权用户才能解锁并且使用当前设备。

​ 传统的用户认证机制包括基于口令的身份认证、质询-响应身份认证、基于生物特征的身份认证和基于位置的身份认证等[1]，在通用个人设备上都可以使用。但是上述认证方式中都存在部分安全漏洞，如果攻击者知道虚拟数字电路板的布局，很容易通过观察手的动作来推断口令从而伪装用户登录；除此之外，“指纹欺诈”和“面部欺诈”技术的发展表示生物特征也可以被成功伪造并且加以利用。

​ 本文提出了基于音乐节奏敲击的身份认证方式，用户通过一段基于音节节拍的敲击节奏来验证自己的身份，敲击节奏只有自己本人知晓。这是一种新式的密码式身份验证，用户在敲击时选择不同的音乐节拍，而不是数字、字母和字符等常用符号。同时，这种认证方式还具有一定的生物特征，每个人对于一段相同的音乐节拍会产生不同的理解和感受，因而不会产生两个完全一样的敲击节奏。综合上述知识和生物特征，这种方式可以与其他认证手段作为双因素身份验证，以确定用户是否合法。

1.2 产品功能

​ 本系统主要由硬件部分和软件部分组成，用于实现用户身份认证，其基本功能如下：

（1）硬件部分——收集用户认证信息

​ 用户在电子设备上的敲击节奏，会引起周围环境中电场变化，使用特定单片机上的传感器收集的电场变化信息，可以作为用户认证密码。

（2）软件部分——处理用户认证消息

​ 为了方便认证，需要将电场变化信息转化成可视化波形，在波形上处理数据。从处理好的数据中提取数据特征作为认证关键信息，录入其数据特征作为后续用户登录时的身份认证。其中有分为两个过程：在注册阶段，用户要求创建自己的敲击节奏，多次输入作为训练样本；在登录阶段，用户只需要输入之前注册的敲击节奏即可解锁设备，如果与训练样本匹配，则认证为授权用户，否则访问请求将被拒绝。

1.3 特色描述

​ 与传统认证方式相比，基于音乐节拍的敲击节奏认证具有相对较好的安全性，它具有以下特色：

​ 第一，可以有效抵御窃取密码的攻击。与传统的密码模式验证身份不同，它不需要用户使用键盘输入字符等信息，只需要用户在需要认证的电子设备上面敲击自己的节奏。由于每个人对于音乐有不同的理解和感受，对手即使观察密码输入过程，也无法完全复现出来。

​ 第二，操作简单。它只需要用户进行敲击节奏，无需费力执行。

​ 第三，双重身份认证更加安全。由于它既包含显性知识，又包含隐形生物特征，因此对于猜测攻击和肩部冲浪攻击等具有鲁棒性。

3.1 硬件测量

​ 本作品的第一个挑战是如何从有噪声的环境中检测并提取出有用的节奏信息。现在常见的节奏检测多是在移动设备上直接检测的，并且用惯性传感器也可以很容易地检测到拍击力度、拍击节奏等信息。但是这种方法的普遍性受到了检测环境或成本的限制。

​ 人的肌肉属于导电体的范畴，人体活动会导致电场变化，当人体与电子设备接触时会产生耦合，会扭曲电子设备附近的电位梯度，所以尝试通过监测周围电场变化来恢复个人拍击节奏。

​ 本作品使用Arduino Nano系列开发板对周围电场进行测量。Arduino Nano系列开发板测量原理如下

#如何测量的每秒2302个数据点？

本作品进行环境噪声检测流程如下：

  1. 设置环境噪声检测时间为m_time，打开端口，开始进行测量；

  2. 由上述可知，每秒可以检测到2302个数据点，利用循环，对Arduino Nano开发板A6端口测量得到的模拟电压值进行m_time×2302次收集，便可得到这段时间内的环境电压数据集合；

  3. 测量结束后，关闭端口；

  4. 调用滤波模块对测量得到的数据集合进行过滤；

  5. 计算过滤后得到数据集合的平均值，即计算环境模拟电压的平均值，即得到环境噪声数据。

3.3数据处理

3.3.1过滤

​ 在现实环境中，各种电子设备产生的电磁波，及各类人类活动，都会使波形产生异形波动，干扰测量。所以我们需要对测量的原始数据进行过滤，减弱环境噪声对实验结果造成的影响。

​ 本实验选用椭圆滤波器进行滤波。椭圆滤波器传递函数如下：

​ $T_{n}(s)$=$\frac{B_{n}(0)}{B_{n}(\frac{s}{\omega_{0}})}$

​ 椭圆滤波器的传递函数为常数除以一个多项式,为全极点网络,仅在无限大阻带处衰减为无限大，在有限频率范围内存在传输零点和极点。极零点在通带内产生等波纹,阻带内的有限传输零点减少了过渡区,可获得极为陡峭的衰减曲线，通带、阻带逼近特性良好，也就是说在阶数相同的条件下,椭圆滤波器相比于其他类型的滤波器，能获得更窄的过渡带宽和较小的阻带波动。在另一方面，对于相同的性能要求，椭圆滤波器相较于巴特沃斯和切比雪夫滤波器，所需的阶数较低，且它的过渡带比较窄。

综上所述，本作品选用椭圆滤波器，将椭圆滤波器运用到本作品的流程步骤如下图所示

椭圆滤波器的振幅平方函数如下：式中$\Omega_{p}$为通带截止角频率，$\Omega_{s}$为阻带截止频率，$\varepsilon$为纹波系数，$R_{N}(x)$为N阶雅可比椭圆函数，$R_{p}$为通带最大衰减，$A_{s}$为阻带最小衰减，$K\left( x \right)$为第一类椭圆积分

​ $\left| H_{a}\left( j\Omega \right) \right|^{2} = \frac{1}{1 + \epsilon^{2}R_{N}^{2}\left( \frac{\Omega}{\Omega_{p}} \right)}$

​ $\varepsilon$=$\sqrt{10^{\frac{R_{p}}{10}} - 1}$

​ $N = \frac{K(k)K(\sqrt{1 - k_{1}^{2}})}{K(k_{1})K(\sqrt{1 - k^{2}})}$

​ $k = \frac{\Omega_{p}}{\Omega_{s}}$

​ $k_{1} = \frac{\varepsilon}{A^{2} - 1}$ $A = 10^{A_{s}}$

​ $K(x) = \int_{0}^{\frac{\pi}{2}}\frac{d\theta}{\sqrt{1 - x^{2}\sin^{2}\theta}}$

3.3.2包络

​ 本作品将过滤后的数据进行包络处理的原因有两点：①原始数据点较为集中，数据集较庞大，处理起来较耗费资源；②增强对信号中混入噪声的抑制能力、减弱周围环境噪声对电场造成的波动对节奏提取的影响。

本作品对已过滤的采集数据进行包络提取的流程如下：

​ 1. 获取一阶差分，并记录差分为0的下标位置

​ 一阶差分为离散函数中连续相邻两项之差，即自变量从x变到x+1时，函数y=y(x)的改变量${\Delta}y_{x} = y_{x + 1} - y_{x}$ 。获取一阶差分为0的点的目的是找到那些平坦的峰值（多于一个等幅宽的样本），将一阶差分值为0的点坐标记录到数组zeros[]。

​ 2. 获取数组zeros的一阶差分

​ 数组zeros中的数据是一阶差分值为0的点的横坐标，所以若数组zeros中相邻两数据的一阶差分值为1，即代表这两点之间是水平波形。获取zeros中一阶差分值不为1的数据坐标，用其对数据zeros进行分割，即可得到每段水平波的起始点与终点。

​ 3. 对每一个水平波进行处理，使得每一个水平波中仅具有一个峰值转向点。

​ 在峰在峰值点检测时，处于水平波中的各个点表现相同。故选取水平波中的中值点作为唯一的峰值点，如果样本数为偶数，则向下舍入。并对一阶差分dy进行修正。

​ 4. 获取符合峰值特性的峰值点

​ 如果数据点符合峰值点判断依据：$\text{dy}\left\lbrack i \right\rbrack > 0,\text{dy}\left\lbrack i + 1 \right\rbrack < 0$，那么点i为信号的峰值点位置。

​ 5. 筛除无效的峰值点

​ 设置相邻峰之间的样本中所需的最小水平距离设置为distance。将步骤四中得到的峰值点，按照其幅值从大到小进行排序，逐个提取峰值点，并以其为中心，±distance为区间，将此区间内的所有点均视为非峰值点，以移除一些较小的无效峰。

​ 6. 将有效峰值进行仿样插值

​ 本作品选用三次样条插值，求解经过有效峰值的曲线，然后根据得到的曲线预测未知位置点。

​ 之所以选用插值而不是拟合有两点原因：①对数据集的不同部分可以拟合出不同的函数，而函数之间的曲线平滑对接；②插值曲线通过所有的已知数据。所以插值可以得到更接近真实情况的包络曲线f。

​ 之所以选用三次样条插值有两点原因：①高次插值可能出现大幅度波动现象，采用分段低次插值可以提高近似程度；②三次及以上多项式可以满足节点处光滑和曲率最小要求，而次数高的曲线容易产生振荡，因而选用三次多项式。

​ 本作品进行三次样条插值的流程如下： ​ （1） 将数据分为n个区间，在每个小区间内的曲线是一个三次样条函数方程： $s_{i}\left( x \right) = a_{i} + b_{i}(x - x_{i}) + c_{i}{(x - x_{i})}^{2} + d_{i}{(x - x_{i})}^{3}$

​ （2） 函数要满足条件1：曲线经过节点

由$s_{i}\left( x \right) = a_{i} + b_{i}(x - x_{i}) + c_{i}{(x - x_{i})}^{2} + d_{i}{(x - x_{i})}^{3}$=$y_{i}$可得$a_{i}$=$y_{i}$ 用$h_{i}$=$x_{i + 1}$-$x_{i}$表示步长，由$S_{i}\left( x_{i + 1} \right) = y_{i + 1}$推出$a_{i} + {h_{i}b}_{i} + {h_{i}}^{2}c_{i} + {{h_{i}}^{3}d}_{i}$=$y_{i + 1}$

​ （3） 函数要满足条件2：曲线一阶导连续（光滑）

​ 由$S_{i}^{'}\left( x_{i + 1} \right) = S_{i + 1}^{'}\left( x_{i + 1} \right)$推出 ​ $S_{i + 1}^{'}\left( x_{i + 1} \right) =b_{i} + {2c}_{i}{h}_{i} + 3d_{i}{h}^{2} $ ​ $S_{i + 1}^{'}\left( x_{i + 1} \right) = b_{i + 1} + {2c}_{i}\left( x_{i + 1} - x_{i + 1} \right) + 3d_{i}\left( x_{i + 1} - x_{i + 1} \right)^{2} = b_{i + 1}$ ​ 可得$b_{i} + {2c}_{i}{h}_{i} + 3d_{i}{h}^{2} = b_{i + 1}$

​ （4） 函数要满足条件3：曲线二阶导连续（曲率最小）

​ 由$S_{i}^{''}\left( x_{i + 1} \right) = S_{i + 1}^{''}\left( x_{i + 1} \right)$可得${2c}_{i} + 6_{i}d_{i} = {2c}_{i + 1}$ ​ 设$m_{i}$=$S_{i}^{''}\left( x_{i} \right) = 2c_{i}$，即$c_{i} = \frac{m_{i}}{2}$，则$m_{i} + 6h_{i}d_{i} = m_{i + 1}$ ​ 可得$d_{i} = \frac{m_{i + 1} - m_{i}}{6h_{i}}$ ​ 将$a_{i}$、$c_{i}$、$d_{i}$代入$a_{i} + {h_{i}b}_{i} + {h_{i}}^{2}c_{i} + {{h_{i}}^{3}d}_{i}$=$y_{i + 1}$可得 ​ $b_{i} = \frac{y_{i + 1} - y_{i}}{h_{i}} - \frac{h_{i}}{2}m_{i} - \frac{h_{i}}{6}(m_{i + 1} - m_{i})$

​ （5） 函数要满足条件4：两端节点满足约束条件

​ 指定端点二阶导数为0，$S^{''}(x_{0}) = 0 = S^{''}(x_{n})$ ​ 即$m_{0} = 0,m_{n} = 0$ ​ 将$a_{i}$、$b_{i}$、$c_{i}$、$d_{i}$带入$b_{i} + {2c}_{i}{h}_{i} + 3d_{i}{h_{i}}^{2} = b_{i + 1}$可得 ​ $$h_{i}m_{i} + 2(h_{i} +h_{i + 1})m_{i + 1} +h_{i + 1}m_{i + 2} = 6\left\lbrack \frac{y_{i + 2} - y_{i + 1}}{h_{i + 1}} - \frac{y_{i + 1} - y_{i}}{h_{i}} \right\rbrack$$

​ 这样可以构造一个以m为未知数的线性方程组

​

可以发现左侧的系数矩阵为严格对角占优矩阵，故线性方程组有唯一解。将方程组解$m_{i}$带入$a_{i}$、$b_{i}$、$c_{i}$、$d_{i}$，便可得到在每个子区间$x_{i} \leq x \leq x_{i + 1}$中的插值函数。

​ （6） 生成数据起点与终点之间的均匀间隔的x值

​ 求出这些x所在区间对应的插值函数值，即得到进行包络处理后的数据。

3.4敲击结束点提取

​ 根据观察波形图可以得到三个观察结果：①一个敲击事件的开始和结束分别对应于开始偏移到返回到基准线的相位差；②当用户刚刚触摸到电子设备时，相位差首先快速增长到局部最大值，然后快速减小到局部最小值，③当用户的手掌从降低压力到完全离开电子设备时，相位差会快速恢复到基准线。

​ 由观察结果可知，有两种情况会导致波形局部方差达到最大：①开始触摸电子设备时，电压值先快速增加，后快速减小；②触摸结束时电压值快速增长。本作品选取触摸结束点的具体实现过程如下：

​ ① 调用环境噪声检测模块，以平均环境噪声值×0.75作为界限，判断低于该界限值的波形有几段，即可得到一段节奏中进行了几次拍击，记为rhythm_number；

​ ②从包络处理后数据的第一个点开始，计算每十个点的方差值；

​ 方差计算公式为$S = \frac{1}{N - 1}\sum_{i = 1}^{N}{(X_{i} - \overset{\overline{}}{X})}^{2}$

​ 其中N=10，$\overset{\overline{}}{X} = \sum_{i = 1}^{N}x_{i}$

​ ③运用与包络提取模块中相同的算法提取上述方差值的峰值，因为敲击结束点所在区间的方差是显著大于其他区间的，所以设置相邻峰之间的样本中所需的最小水平距离为800；

​ ④ 对方差峰值从大到小进行排序，找出方差峰值中的前rhythm_number×2个峰值，其中第奇数个方差峰值为触摸点，第偶数个方差峰值为抬手点；

​ ⑤ 将偶数个方差峰值对应的索引值保存下来，即可得到每次敲击时的抬手点时刻。

3.5敲击开始点提取

由3.4中对波形观察结果的分析可知，每次开始触摸时，电压值会快速增大后，立即快速减小，通过对不同用户、不同节奏类型敲击波形的分析与对比，可以得到结论：不同用户每次触摸电子设备时产生的波形变化是类似的。

所以可以将一段开始触摸，接触电子设备时，电压值从快速增长到快速降低的这一段波形保存下来，作为“经验对照曲线”。

在一段敲击节奏中，以100作为起始点间隔，2500作为数据长度，将数据划分为n段，在其中找出与“经验对照曲线”相似度最高的rhythm_number段曲线，这rhythm_number段节奏的起始点保存下来，即可得到每次敲击时的起始时刻。

本作品敲击开始点提取模块效果如下，彩色曲线为与“经验对照曲线”相似度最高的曲线，曲线几点对应的横坐标即为拍击开始时刻。

3.6分类

将分类算法进行划分，根据类别个数的不同可以分为单分类、二分类、多分类。本作品中用户类别分为：合法用户、非法用户，而合法用户数量是无限的、可变化的，且非法用户节奏特征是未知的。如果选用二分类算法，分为合法用户和非法用户两类，假如出现新的非法用户试图进行认证，非法节奏输入训练好的分类器进行预测，它的节奏无法被准确分类为两类中的任意一类，因为它的节奏特征既不与合法用户特征相似，也不与分类器原先训练过的非法用户特征相似，解决这种情况的一种方法是对所有错误状态进行模拟，但这是不现实的，并且代价是很昂贵的。这时候选用单分类器可以很好的解决这种样本类别数据不平衡的问题，单分类器中，仅仅只有一类的信息用于训练，算法思路就是寻找一个最优超平面，将这类数据全部包含在里面，预测就是用这个超平面做决策，在圈内的样本就认为是正样本。

单分类的算法思想是在特征空间中获得数据周围的球形边界，并且希望最小化这个超球体的体积，从而最小化异常点数据的影响。

算法实现需要定义的最小化的目标函数如下，其中a为形状的中心，R表示半径。与传统支持向量机类似，可以要求所有训练数据点$x_{i}$到中心的距离严格小于R，但是同时构造一个惩罚系数为C的松弛变量$\xi_{i}$

min $F\left( R,a,\xi_{i} \right) = R^{2} + C\sum_{i = 1}^{m}\xi_{i}$

s.t. $\left( x_{i} - a \right)^{T}\left( x_{i} - a \right) \leq R^{2} + \xi_{i}$ $\xi_{i} \geq 0$

可以看到这是一个凸优化问题，求解该目标函数的全局最优值是十分困难的，算法可能会被困在局部最优点动弹不得，无法收敛到全局最优。可以采用拉格朗日对偶手段，将一个最优化问题转化成另一个更容易求解的对偶问题，将带约束的目标函数转化为不带约束或约束较简单的凸优化问题。

第一步先按照拉格朗日乘数法构造拉格朗日函数,其中$\gamma_{i}$和$\alpha_{i}$为标量—朗格朗日乘子。式子中第一项是原优化问题的目标函数，第二项是所有不等式约束的线性组合，第三项是所有等式约束的线性组合。

$$L\left( R,a,\alpha_{i},\xi_{i} \right) = R^{2} + C\sum_{i = 1}^{N}\xi_{i} - \sum_{i = 1}^{N}\gamma_{i}\xi_{i} - \sum_{i = 1}^{N}{\alpha_{i}(R^{2} + \xi_{i} - \left( x_{i} - a \right)^{T}\left( x_{i} - a \right))}$$

对参数进行求导并令倒数等于0得到：

$$\sum_{i = 1}^{N}\alpha_{i} = 1$$

$$a = \frac{\sum_{i = 1}^{N}{\alpha_{i}x_{i}}}{\sum_{i = 1}^{N}\alpha_{i}} = \sum_{i = 1}^{N}\alpha_{i}x_{i}$$

$$C - \alpha_{i} - \gamma_{i} = 0$$

第二步是构造拉格朗日对偶函数及对偶问题。因为导致优化问题不一定是凸问题的原因是目标函数，所以试图摆脱自变量R,a，即可将问题转换为容易进行研究的凸问题。将上述式子带入拉格朗日函数，可得到如下拉格朗日对偶函数：

$$L = \sum_{i = 1}^{N}{\alpha_{i}(x_{i}^{T}} \bullet x_{i}) - \sum_{i,j = 1}^{N}{\alpha_{i}\alpha_{j}(x_{i}^{T}} \bullet x_{j})$$

第三步是使用核函数，用低维输入空间的函数$K(x_{i},x_{i})$代替在高位空间的内积${(x}{i}^{T} \bullet x{i})$，这样向量机就不需要进行复杂的非线性变化，大大简化计算。

$$L = \sum_{i = 1}^{N}{\alpha_{i}K(x_{i},x_{i})} - \sum_{i,j = 1}^{N}{\alpha_{i}\alpha_{j}K(x_{i},x_{j})}$$

训练结束后，判断一个新的数据点Z是否是这个类，那么就看这个数据点是否在训练出来的超球面里面，如果在里面，即$\left( z - a \right)^{T}\left( z - a \right) \leq R^{2}$，则判定为属于这个类，将超球面的中心用支持向量来表示，最终判别函数如下所示，输出为1代表属于目标类，输出为-1代表是异常数据。

$$K\left( z,z \right) - 2\sum_{i = 1}^{N}{\alpha_{i}K\left( z,x_{i} \right)} + \sum_{i,j = 1}^{N}{\alpha_{i}\alpha_{j}K\left( x_{i},x_{j} \right) \leq R^{2}}$$

本作品选用高斯核函数，即$K\left( x,x^{‘} \right) = e^{- \frac{\left| x - x^{‘} \right|^{2}}{2\sigma^{2}}}$

本小组使用二维数据对单分类器性能进行检验，可以看到分类曲面并未将所有训练数据都包含在内，这是上述公式中松弛变量$\xi_{i}$的作用结果，每个数据点都有对应的松弛变量，如果大部分数据点在一个小区域内，少数异常点在离它们很远的地方，为了避免模型为了迎合少数点做出过大牺牲，从而导致过拟合，单分类器选择给离群点一些松弛空间，具有较高的鲁棒性。由此可以看到单分类器的第二个优点：模型不会被个别极端数据点破坏，且模型对离群点很敏感。

第四章 创新性说明

​ 相比较于传统的双重认证，比如人脸认证、指纹认证和密码认证等，敲击认证具有三个尤为突出的创新之处，使得这种认证方式具有更高的安全性，愈发具有商业研发价值，被越来越多的企业和科研工作者关注。

  1. 原理创新。敲击认证是通过敲击某一件物品改变周围环境中的电场，然后通过一个设备收集环境中电场的变化，再将电场变化进行特征提取，就可以得到用户专属的认证信息进行认证。基于原理，与传统的认证方式相比，敲击认证具有更加安全的认证环境，在技术方面更加难以破解。如果在敲击认证过程中被黑客观察并且模仿，黑客也无法百分之百正确且完整地重现敲击节奏，这使得敲击认证具有无与伦比的安全因素。
  2. 价格低廉。通过以上的原理，敲击认证过程中只需要一个制作简单并且造价低廉的设备收集周围电场的变化。这个设备其实是一个单片机，不需要复杂的电路，不需要昂贵的器件，可以进行批发生产。由于成本低廉，不需要投入过多的资本便可以得到极高的回报，敲击认证成本大大降低，越来越多的企业加入其中研究，因此敲击认证具有无限潜力的前景未来。
  3. 具有良好的可移植性。传统的认证都需要一个特定的设备来收集认证信息，比如指纹识别需要特定的指纹采集器采集指纹，人脸识别需要电子摄像头采集人脸信息，密码识别需要一个键盘输入密码信息……这些认证方式都被局限在一定的范围内，但是敲击认证在这个方面具有无与伦比的优势，不管是电子屏幕，还是常见的音箱，甚至是一张常见的桌子，都可以作为敲击的位置进行认证，这种特点使得敲击认证具有极高的普及性，能够在更加广泛的范围内使用。

第五章 总结

​ 随着虚拟现实设备越来越多地融入我们的日常生活，为这些设备获取或访问的数据提供安全性变得至关重要。在本研究中，我们提出了一种基于音乐节拍的敲击节奏的双因素用户身份验证机制，该机制利用用户设计的敲击节奏和节奏差异性的独特生物特征来识别合法用户。与之前的工作相比，我们的解决方案提供了安全的身份验证、较低的认知开销，并提供了极大的便利。我们从时间消耗、登录尝试、用户动作的影响和可记忆性等方面测试了它对各种类型的攻击者的鲁棒性及其实用性，从而进一步衡量了它的安全性。我们观察到，基于音乐节拍的敲击节奏身份认证需要相对较长的注册时间，是因为许多参与者在虚拟现实设备上验证自己的经验有限。随着用户在方案实施后每天多次练习，这种情况可能会得到缓解。此外，由于每个用户只执行一次注册，从长远来看，较长的注册时间不会产生明显的开销。总之，我们相信基于音乐节拍的敲击节奏的双因素用户身份验证机制是一种适用于当前VR设备的实用认证方法。
